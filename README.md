TaskFlow is a simple web-based application built using Python and Flask that helps users manage tasks, deadlines, and task status.

- Add, edit, and delete tasks
- Track task priority and deadlines
- Update task status (Pending, In Progress, Completed)
- Persistent storage using SQLite

## Technologies Used
- Python
- Flask
- SQLite
- HTML & CSS

## Project Purpose
This project was built to practice backend development concepts, database operations, and clean code structure while solving a real-world task management problem.
